# Pomodoro Timer Mac

A small menu-bar Pomodoro timer for macOS.

## Install
1. Download the latest DMG from GitHub Releases.
2. Open the DMG.
3. Drag **Pomodoro Timer** into Applications.
4. Launch it from Applications.

## Behavior
- Lives in the macOS menu bar.
- Shows today's total focus time in the main window.
- Uses 25-minute Pomodoro cycles in the status bar.
- Auto-starts on login.

## First launch note
This project is not code-signed or notarized, so macOS may warn that the app is from an unidentified developer. Use the standard macOS security override on first launch.
